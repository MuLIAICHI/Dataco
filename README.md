## Two-Phase Architecture

### Phase 1 — Validation Notebook (`notebooks/`)
Full batch pipeline following Géron Chapter 2:
- Stratified train/test split on `city`
- Custom `MoroccanFeatureAdder` transformer (ratio features)
- `ColumnTransformer` with median imputation + OHE + StandardScaler
- 3 models: Linear Regression, Decision Tree, Random Forest
- 5-fold cross-validation + `RandomizedSearchCV`

### Phase 2 — Online Learning (`src/online_learning.py`)
Incremental pipeline using `SGDRegressor.partial_fit()`:
- Pulls new listings from Apify Dataset via API (offset-based, no duplicates)
- Preprocessor frozen after cold start — never re-fit
- Model updates incrementally on each new batch
- `--benchmark` flag compares online model vs. Random Forest on a held-out test set

## Quickstart

### 1. Install dependencies
```bash
pip install numpy pandas scikit-learn matplotlib joblib requests
```

### 2. Run the validation notebook
Open `notebooks/morocco_housing_ml.ipynb` in Jupyter.
The notebook expects `data/dataset_mubawab-housing.csv` to exist.

### 3. Run online learning (first time)
```bash
cd src

# Dev mode — uses local CSV
python online_learning.py --local-csv ../data/dataset_mubawab-housing.csv

# Production — pulls from Apify (set your token in the config section first)
python online_learning.py
```

### 4. Incremental update (each new Apify run)
```bash
python online_learning.py
# Automatically picks up from last offset — only trains on new rows
```

### 5. Benchmark online model vs Random Forest
```bash
python online_learning.py --benchmark --local-csv ../data/dataset_mubawab-housing.csv
```

### 6. Force full reset
```bash
python online_learning.py --reset
```

## Configuration

Before running in production, edit the config block at the top of `src/online_learning.py`:

```python
APIFY_API_TOKEN  = "apify_api_xxxx..."   # your Apify token
APIFY_DATASET_ID = "mubawab-housing"     # your dataset name or ID
```

**Never commit your API token.** Add it via environment variable or a `.env` file (see `.gitignore`).
